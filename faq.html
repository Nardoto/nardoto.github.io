Tire suas duvidas sobre o Nardoto Browser, Studio e Extensões

Nardoto Browser NAVEGADOR

O que e o Nardoto Browser?

E um navegador baseado no Chrome que eu criei do zero, com todas as 11 extensões ja instaladas e configuradas.

Você baixa, instala, e ja pode usar tudo. Sem precisar instalar extensão por extensão.

Funciona no Mac?

O Nardoto Browser e apenas para Windows por enquanto.

Mas as extensões funcionam no Chrome de qualquer sistema (Windows, Mac, Linux).

As extensões atualizam automaticamente?

Sim! Tanto o navegador quanto as extensões se atualizam automaticamente quando você abre o browser.

Você nunca precisa baixar nada manualmente depois da instalacao.

O navegador e seguro?

Sim! E baseado no Chromium (mesmo motor do Chrome), mas sem rastreamento de dados da Google.

Seus dados são seus. Nao coletamos nenhuma informacao de navegacao.

Posso usar as extensões no Chrome normal?

Pode! Você pode usar as extensões no seu Chrome normal tambem. Basta instalar manualmente.

O navegador Nardoto e so uma opcao mais pratica (tudo instalado + atualizacoes automaticas).

Nardoto Studio APP DESKTOP

O que e o Nardoto Studio?

E um app para Windows que automatiza a edição de vídeos no CapCut Desktop.

Sincroniza legendas, TTS, musica e monta timelines automaticamente.

Preciso do CapCut Desktop instalado?

Sim! O Nardoto Studio trabalha em conjunto com o CapCut Desktop.

Ele automatiza tarefas dentro do CapCut, entao você precisa ter o CapCut instalado.

Funciona no Mac?

Nao, o Nardoto Studio e apenas para Windows.

Ele depende de automacao de interface que so funciona no Windows atualmente.

Qual a diferença entre Studio e as extensões?

Extensões: Funcionam no navegador, automatizam sites como VEO3, Suno, ChatGPT.

Studio: Funciona no desktop, automatiza o CapCut para edição de vídeo.

São produtos complementares - extensões geram conteudo, Studio edita vídeos.

Como funciona a assinatura?

Todos os produtos são assinaturas mensais via Stripe.

Você pode cancelar a qualquer momento sem multas ou taxas.

Extensões Chrome 11 EXTENSOES

O que são as extensões Nardoto?

São extensões para Google Chrome que automatizam tarefas repetitivas em ferramentas de IA como VEO3, Wisk, Suno, CapCut, ChatGPT, Claude e mais.

Em vez de colar prompts um por um, você cola uma lista e a extensão envia todos automaticamente. Downloads tambem são feitos em massa.

Quais ferramentas/sites as extensões funcionam?

Geração de vídeo: VEO3 (Google), Wisk

Geração de musica: Suno AI

Edicao: CapCut (web)

IAs de texto: ChatGPT, Claude, Gemini

Geração de imagem: Midjourney

Voz: Google TTS

Preciso ter conta nas ferramentas (VEO3, Suno, etc)?

Sim! As extensões automatizam o uso das ferramentas, mas você precisa ter sua propria conta em cada uma delas.

Exemplo: para usar a extensão do VEO3, você precisa de acesso ao VEO3 no Google AI Studio.

Como instalo as extensões?

Opcao 1 (Recomendada): Baixar o Nardoto Browser - ja vem tudo instalado.

Opcao 2: Instalar manualmente no Chrome via arquivo .zip.

Com assinatura ativa, você recebe acesso aos arquivos de instalacao.
