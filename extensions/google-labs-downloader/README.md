# veo3-clicker

Estrutura de arquivos para uma extensão que simula cliques no site veo3 para baixar vídeos automaticamente.

Observação: nenhum arquivo contém código — esta pasta contém apenas a estrutura (placeholders) conforme solicitado.

Estrutura proposta:
- manifest.json (placeholder)
- src/
  - content.js (placeholder)
  - background.js (placeholder)
  - popup.js (placeholder)
  - popup.html (placeholder)
  - styles.css (placeholder)
- icons/ (placeholders para ícones)
- docs/ (instruções e notas de implementação)
- scripts/ (scripts de build/empacotamento vazios)
- tests/ (esboço de testes)
- dist/ (build output placeholder)

Use esta estrutura como ponto de partida para implementar a extensão. Lembre-se de respeitar os termos de serviço do site alvo e obter permissões necessárias.
