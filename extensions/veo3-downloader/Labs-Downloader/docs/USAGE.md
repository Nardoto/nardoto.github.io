# Como usar a extensão veo3-clicker

1. Instale a extensão no Chrome:
   - Abra chrome://extensions/
   - Ative o "Modo do desenvolvedor"
   - Clique em "Carregar sem compactação"
   - Selecione a pasta `veo3-clicker`

2. Use a extensão:
   - Vá para qualquer página do veo3 com um vídeo
   - Um botão verde "⬇️ Baixar Video" aparecerá no canto inferior direito
   - Clique no botão para baixar automaticamente na qualidade original (720p)

## Funcionamento
A extensão adiciona um botão flutuante que, quando clicado:
1. Localiza o botão de download da página
2. Simula o mouse hover para mostrar o menu
3. Encontra e clica na opção "Tamanho original (720p)"
4. Inicia o download automaticamente

## Resolução de Problemas
Se o download não iniciar:
1. Verifique se o vídeo está carregado na página
2. Recarregue a página e tente novamente
3. Verifique o console do desenvolvedor para mensagens de erro
