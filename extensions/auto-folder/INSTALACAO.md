# Guia de Instalação Rápida

## Extensão: Abrir Múltiplas Abas

### Passo a passo para instalar:

#### 1. Abrir a página de extensões do Chrome
- Abra o Google Chrome
- Digite na barra de endereços: `chrome://extensions/`
- Pressione Enter

#### 2. Ativar o Modo do Desenvolvedor
- No canto superior direito da página, encontre o toggle "Modo do desenvolvedor"
- Clique para ativar (deve ficar azul/ativado)

#### 3. Carregar a extensão
- Clique no botão "Carregar sem compactação" (ou "Load unpacked")
- Navegue até a pasta: `c:\Users\tharc\Videos\extenções\autopasta`
- Clique em "Selecionar pasta"

#### 4. Pronto!
- A extensão "Abrir Múltiplas Abas" aparecerá na lista
- Você verá o ícone roxo com abas na barra de ferramentas do Chrome
- Se não aparecer, clique no ícone de extensões (quebra-cabeça) e fixe a extensão

### Como usar a extensão:

1. Clique no ícone da extensão na barra de ferramentas
2. Digite a URL completa do site (ex: https://www.google.com)
3. Digite quantas abas deseja abrir (máximo 20)
4. Clique em "Abrir Abas"
5. As abas serão abertas instantaneamente!

### Recursos:

- Interface bonita e moderna
- Validação de URLs
- Limite de segurança (20 abas)
- Feedback visual
- Suporte a tecla Enter

### Solução de problemas:

**A extensão não aparece:**
- Verifique se o "Modo do desenvolvedor" está ativado
- Certifique-se de selecionar a pasta correta

**Erro ao carregar:**
- Verifique se todos os arquivos estão na pasta
- Recarregue a extensão clicando no ícone de refresh

**Os ícones não aparecem:**
- Os ícones já estão na pasta `icons/`
- Se necessário, recarregue a extensão
