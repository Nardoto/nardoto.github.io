Baixar Nardoto Studio para Mac

Apple Silicon (arm64) · ~420 MB

Recomendado
Script automático
Um comando faz tudo: baixa, libera o bloqueio do macOS e abre o instalador.
1
Abra o Terminal — pressione Cmd + Espaço, digite "Terminal" e pressione Enter.
2
Cole o comando abaixo e pressione Enter.
3
Aguarde o download (~420 MB) e siga o instalador que abrir.
Terminal
curl -L "https://github.com/LoopLess-nardoto/nardoto-studio-releases/releases/latest/download/NardotoStudio-Setup-mac.dmg" -o ~/Downloads/NardotoStudio-mac.dmg && xattr -cr ~/Downloads/NardotoStudio-mac.dmg && open ~/Downloads/NardotoStudio-mac.dmg
Alternativa
Instalação manual
Baixe o DMG e siga os passos para liberar o app no macOS.
1
Baixe o instalador clicando no botão abaixo.
2
Abra o DMG e arraste o Nardoto Studio para a pasta Aplicativos.
3
Se aparecer "app danificado", abra o Terminal e execute o comando abaixo.
4
Se ainda bloquear: Preferências do Sistema → Privacidade e Segurança → "Abrir mesmo assim".
Baixar DMG (~420 MB)
Depois, se precisar
Terminal
xattr -cr "/Applications/Nardoto Studio.app"
Por que o macOS bloqueia? O Nardoto Studio ainda não possui assinatura digital Apple (requer conta Apple Developer de $99/ano). O aviso não indica risco — o app é seguro. Este processo será eliminado em uma versão futura com code signing.
