Veja na prática

Do zero ao vídeo pronto

Acompanhe o fluxo completo de produção de um vídeo usando o Nardoto Studio.

1

Criar workspace e projeto

Abra o app, crie seu workspace e insira um projeto do CapCut para começar a produção.

2

Gerar roteiro com IA

Use o Claude Code no chat com botões de prompts personalizados para gerar o roteiro do vídeo.

3

Verificar documentos gerados

Abra a árvore de arquivos para conferir todos os documentos que o chat gerou automaticamente.

4

Gerar legendas SRT

Crie as legendas SRT a partir do roteiro e abra o CapCut para conferir como ficaram.

5

Gerar áudio no CapCut

O Studio gera o áudio e envia direto para o CapCut. Sem etapas manuais.

6

Visualizar preview

Confira o resultado parcial antes de continuar. Preview rápido direto no app.

7

Gerar prompts de imagens

Use o chat para gerar os prompts e criar as imagens que ilustrarão cada cena do vídeo.

8

IA gerando imagens

Acompanhe o chat respondendo e gerando cada prompt de imagem automaticamente.

9

Gerar imagens no Flow

Abra o Flow para gerar as imagens a partir do documento de prompts, usando VEO3 ou outras IAs.

10

Inserir músicas

Pesquise músicas no Epidemic Sound e insira diretamente na timeline do projeto.

11

Sincronizar mídias

Sincronize as mídias com as legendas e o áudio. Tudo alinhado automaticamente.

12

Conferir no CapCut

Abra o CapCut e confira como ficaram todos os arquivos editados. Vídeo pronto.
